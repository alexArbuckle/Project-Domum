## Project Domum
> Control an arbitrary amount of relays with a Discord bot.

### Requirements
```
Raspberry Pi 4
Ubuntu Server 20.04
```

### Installation
```
pip3 install RPi
pip3 install suntime
pip3 install discord.py
sudo apt-get -y install python3-rpi.gpio

git clone https://github.com/lxRbckl/Project-Domum.git
```
